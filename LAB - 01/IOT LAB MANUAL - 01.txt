Arduino IDE

2. Create a first program using Arduino IDE with Hardware Interface and without hardware 
interface. 
3. Write a program to perform an Arithmetic operation.

9. Write a program to print the first 50 even numbers. 
10. Write a program to print numbers in reverse order.
